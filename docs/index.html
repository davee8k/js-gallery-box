GalleryBox demos

You can use ←, → for paging and Esc for closing gallery window.

Basic link usage

$("#gallery1").galleryBox({});

Use only specific class

$("#gallery2").galleryBox({page: '.gallery-link'});

This is text with link on gallery image one, next link without gallerybox and another onegallery image three.

Without background

$("#gallery3").galleryBox({background: false});

Disable paging between images

$("#gallery4").galleryBox({arrows: false});

Show image counter

$("#gallery5").galleryBox({pager: true});
